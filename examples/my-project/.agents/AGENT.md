# Agent Rules — My Project

Project-specific behavioral rules injected into every agent session.

## General

- Ask clarifying questions before starting large tasks.
- One task at a time; complete and verify before moving to the next.

## Coding

- Handle errors explicitly; never swallow exceptions silently.
- Prefer idiomatic patterns over one-off workarounds.

## Testing

- Run tests after every functional change.
- A change is not complete until tests pass.
