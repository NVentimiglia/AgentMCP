# Sessions

Raw conversation logs. One file per session.

## Naming
YYYY-MM-DD-topic.md  (e.g. 2026-05-14-react-hooks.md)

## How to record
- Claude.ai: Export conversation → paste here
- Cursor: Copy conversation history → paste here
- Gemini: Export → paste here
- Any agent: copy-paste the raw exchange is fine

## Format
No required format. Paste the raw exchange as-is.
The learn pass handles extraction.

## What to record
Record sessions where you:
- Solved a non-obvious problem
- Corrected an agent mistake
- Discovered a better pattern
- Wrote a prompt that worked well

Skip sessions that are purely mechanical.
